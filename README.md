### Gamepad Controls
- Menu:
    - **Up**: Dpad Up
    - **Down**: Dpad Down
    - **Select**: A
    - **Back**: B
- Game:
    - **Move**: Left Stick
    - **Run**: Left Bumper
    - **Jump**: A
    - **Attack**: Y
    - **Aim**: Right Stick
    - **Posess**: Right Bumper

### Keyboard Controls
- Menu:
    - **Up**: Up
    - **Down**:  Down
    - **Select**: Enter
    - **Back**: Backspace
- Game
    - **Move**: Left/Right
    - **Run**: Left Shift
    - **Jump**: Spacebar
    - **Attack**: W
    - **Aim**: *Not implemented
    - **Posess**: *Not Implemented
